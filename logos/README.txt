REAL COMPANY LOGOS
==================

Drop the real logo image for any company in this folder using the EXACT
filename below. The site will auto-detect it on the next page reload and
swap the text badge (e.g. "MS", "PMI") with the real logo.

Recommended:
- Format: PNG with transparent background (or SVG converted to PNG)
- Size:   square, ~256x256 px (it will be displayed at 56-80 px)

Notes
-----
- If a file is missing or fails to load, the site falls back to the
  styled text badge automatically — nothing breaks.
- The lookup is done with a HEAD request once at page load, so just
  refresh the page after dropping new logos in.
